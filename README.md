# 一键加验证特点

## 1.无需编写代码

仅需一个exe即可自动添加超高强度防破解的网络验证

## 2.高强度加密exe

会抽取您的exe代码进行虚拟化并加密（使用VMProtectBeginVirtualizationLockByKey(),此API会将您的代码进行虚拟化并加密，只有设置了正确的VMP授权，代码才会正常解密，建议您在自己程序中使用VMProtectBeginVirtualizationLockByKey/VMProtectEnd来更精准的保护您自己的代码）

## 3.无卡密，无代码

只有卡密登录成功后，后台才会动态生成VMP授权，此时被虚拟化并加密的代码才会解密运行。即使网络验证被强制跳过，exe功能也无法运行！

## 4.一次一密

每个数据包都有自己唯一nonce值和有效期，重复使用同一数据包会直接会判定为非法请求，完全杜绝重放攻击、山寨、本地模拟等

## 5.强大反调试

内置了多种检测调试的技术手段，一但检测到程序正在被调试，会立即拉黑卡密、ip和机器码（可选）
